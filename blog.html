Bridging Model Context Protocol with Network Orchestration

Aug 12, 2025 • — • AI · Networking

How I wired an AI assistant through a custom MCP server to speak RESTCONF with Cisco NSO & keep hallucinations out of the control plane.

Collapse

Why: LLMs are great at reasoning around desired state but untrusted at executing imperative network changes. The Model Context Protocol (MCP) lets you expose only safe, well-typed actions.

I built an adapter that reflects NSO RESTCONF endpoints into MCP tools. Each tool declares input schema & pre-flight validation. Returned output is normalized JSON—not raw CLI text.

// Simplified tool descriptor
interface Tool {
  name: string;
  input: Schema;
  run(args): Promise;
}
// Example: fetch service instance
tool("get_service", { service: string }, async ({service}) => nsoreq(`/restconf/data/svc:${'{'}service{'}'}`));

Deterministic envelopes + constrained side-effects give the model guardrails while preserving flexible goal reasoning.

  • Map: RESTCONF -> canonical tool set
  • Add auth + rate limits
  • Cache read-only queries
  • Require idempotency proof for mutating ops

Result: zero unaudited config diffs; faster intent translation; human trust improved because every action is logged with structured parameters.

Next: introduce a diff simulation tool & integrate policy checks before write operations.

Hard Lessons from Small Reinforcement Learning Runs

Jul 2, 2025 • — • AI

Micro-budgets force discipline: reward shaping sanity checks, variance tracking & deliberately tiny experiments.

Read more

Instead of scaling prematurely, I treated every run as a falsifiable hypothesis. If a change didn’t tighten variance bands in 3 trials, it was discarded.

Signal quality beats parameter count. I added a console.table()-driven dashboard during training printing rolling KL, entropy & episodic return deltas.

// Pseudo logging snippet
logStep({
  step,
  return: epReturn,
  entropy: policy.entropy(),
  kl: klDiv,
  lr: optimizer.lr
});

That surfaced early collapse in one variant long before aggregate plots would have. Most “improvements” were noise.

Rule of thumb: if a tweak needs >5 seeds to look positive, it's probably placebo.
