Bridging Model Context Protocol with Network Orchestration

Aug 12, 2025 • AI · Networking

Problem: LLMs can reason about target network states but shouldn't be given raw imperative levers.

Approach: Reflect a curated allow‑list of NSO RESTCONF endpoints into structured MCP tools: JSON schema, idempotency, validator, logging.

tool("get_service", { service: string }, async ({service}) => nsoreq(`/restconf/data/svc/${service}`));
  • Canonical tool naming for prompt stability.
  • Dry-run diff + approval flag before mutation.
  • Caching of read ops (shape hash).
  • Structured audit log for each call.

Outcome: Faster intent → safe action, zero unaudited diffs.
